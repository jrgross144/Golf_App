Welcome to the Golf Scorecard Application

It allows golfers to keep track of how well they play at different courses and with different groups

Author: Jacob Gross


Useful Features

  • Outings can be made for multiple people and multiple scorecards
  • Average and winners are calculated for outings
  • Golfer averages are displayed along side par, yards, course rating and course slope
  • Golfers can see their handicap and have it updated after each round is entered in
  • Tests were written to make sure that each of the calculations above were made properly
  • The site will also keep track of the most recent course, user and outing you have viewed/created so that you can simply fill out forms

Directions for the happy path

  1. Go to the profile selection and select a user
  2. Go to outings and create an outing
  3. Go to courses and select show for one of the courses
  4. Select Add New Scorecard at the bottom of the table (The user and course ids should be automatically filled in for you)
  5. Enter in score and any other data that you would like to change
  6. Go to the course and see that the statistics for user average changed
  7. Go to the user and see that the statistics for handicap changed
  8. (Optional) Enter in more scorecards
  9. Review the outing on the outing page and see the average score, winner, the course, date and related scorecards
